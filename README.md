# QBP
Template-based project generator for Qt apps.

Uses [Breeze](http://api.kde.org/frameworks/breeze-icons/html/) icons by the [KDE® Community](http://www.kde.org/).

# Features
Currently supports only 2 templates:
* Qt Widgets
* Qt console app

# Usage
`./QBP.bash`
